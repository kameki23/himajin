FEATURES

こんな使い方ができます

全国の「今ヒマな人」を一覧表示。都道府県・年齢・ヒマ度で気の合いそうな人を見つけられます。

SAFETY

安心して使える仕組み

みんなが気持ちよく雑談できるよう、安全対策に力を入れています。

✓
18歳以上だけ利用開始時に年齢確認。18歳未満の方は利用できません。
✓
出会い・性的目的は禁止規約で明確に禁止。違反は即時アカウント停止の対象です。
✓
24時間365日の監視AIによる自動コンテンツ監視と運営によるモニタリング体制。
✓
ブロック・通報機能全ての画面にブロック・通報を搭載。気になる相手はすぐ対処。
✓
違反ユーザーは自動退場通報が一定数を超えると自動でアカウントを停止します。
✓
匿名だから安心電話番号・メール・本名は不要。いつでもワンタップで退会できます。
